# 星火智岗 — 前端页面设计

> 三大核心板块：岗位查询 | AI 简历对话分析 | 大学生职业规划

---

## 技术选型

| 层 | 方案 |
|---|------|
| 框架 | React 18 + TypeScript |
| 路由 | React Router v6 |
| 状态 | Zustand |
| UI | Ant Design 5 |
| 可视化 | ECharts / D3.js |
| 请求 | React Query + Axios |
| 构建 | Vite |

---

```

```
