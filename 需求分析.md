# TalentOS - 星火智岗前端功能



---

## 一、总览

顶部导航栏左侧的首页仪表盘。

### 功能点
| 功能 | 说明 |
|------|------|
| 4个统计卡片 | 累计采集数据、已发现岗位、技能关系数、匹配准确率 |
| 新岗位发现趋势图 | ECharts折线图，展示1-6月新岗位数量和JD总量变化 |
| 数据源覆盖图 | ECharts环形饼图，展示TOP 8数据源分布（BOSS直聘/智联/GitHub等） |
| 技能演化动态流 | 时间线列表，展示技能新增/淘汰/升级变动（如"Java工程师新增MCP"） |

---

## 二、岗位发现中心

### 功能点
| 功能 | 说明 |
|------|------|
| 数据源标签云 | 展示16个数据源：BOSS直聘、智联招聘、猎聘、拉勾、GitHub、HuggingFace、arXiv、知乎、CSDN、开源中国、国家政策、教育部目录、AI日报、技术博客、GitHub Trending、PapersWithCode |
| 新岗位增长柱状图 | 7个新兴岗位（AI Agent架构师、Agent Workflow工程师等）按增长率排列 |
| LLM智能聚类结果 | 岗位卡片展示：名称、描述、增长率、需求等级、薪资范围、引用来源 |
| 实时JD采集流 | 15条岗位JD列表，含编号、岗位名、技能标签、公司来源 |

### 静态数据
- 16个数据源
- 7个新兴岗位
- 15条实时JD

## 三、能力图谱中心

### 功能点
| 功能 | 说明 |
|------|------|
| 图谱搜索 | 输入框搜索岗位或技能关键词，刷新图谱 |
| 力导向知识图谱 | ECharts图可视化，7个岗位+20个技能节点的力导向布局，支持拖拽缩放 |
| 技能动态演化 | 对比2024年/2026年Java工程师技能集，新增技能标绿、淘汰技能标红 |
| 变化证据引用 | 时间线展示5条证据：GitHub Trending、arXiv论文、BOSS直聘数据、AI日报、Google Trends |

### 演化数据示例
- **2024年** Java工程师：SpringBoot、MySQL、Redis、Docker、MyBatis、Hibernate
- **2026年** Java工程师：SpringBoot、MySQL、Redis、Docker、MCP、AI Agent、RAG、LangChain、K8s、向量数据库

## 四、AI能力森林

### 功能点
| 功能 | 说明 |
|------|------|
| 能力树搜索 | 输入框搜索技能名，实时筛选树节点 |
| 可展开/收起能力树 | 6大能力域、30+技能节点的层级树，点击箭头展开子节点 |
| 径向依赖图 | ECharts径向图展示"AI Agent开发工程师"从根节点到具体技能的依赖关系 |

### 能力树结构（6大域）
| 能力域 | 包含技能 |
|--------|---------|
| 编程语言 | Python（异步编程、类型注解、装饰器）、TypeScript（Node.js、Deno） |
| 大模型技术 | LLM原理（Transformer、Attention、RLHF）、Prompt Engineering（Few-shot、CoT、ToT）、Fine-tuning（LoRA、QLoRA、DPO） |
| Agent框架 | LangChain（LCEL、Tool Calling、Memory）、CrewAI、AutoGPT、MCP |
| 知识增强 | RAG（向量检索、文档切片、重排序）、GraphRAG（知识图谱、图遍历、Neo4j）、向量数据库（Milvus、Faiss、Pinecone） |
| AI部署与运维 | Docker、Kubernetes（GPU调度）、模型服务（vLLM、Triton、ONNX） |
| 工具与平台 | Git、HuggingFace、Weights & Biases |

## 五、AI简历诊断中心

### 功能点
| 功能 | 说明 |
|------|------|
| 文件上传 | 支持点击上传和拖拽上传，接受PDF/Word/JPG/PNG格式 |
| 文件类型识别 | 自动显示文件图标、文件名、文件大小，模拟解析动画（1.5秒） |
| 模拟解析 | "模拟解析"按钮随机加载一份样本简历 |
| 示例简历 | 内置"张三简历"、"李雪简历"2份示例，一键加载 |
| 诊断报告 | 双栏布局：左侧显示姓名/电话/邮箱/学历/经验/推荐岗位，右侧显示技能评分条形图（8项技能） |
| 技能雷达图 | ECharts雷达图，对比个人技能与岗位要求（虚线） |
| 差距分析 | 针对推荐岗位展示每项技能的当前值/目标值进度条，底部给出优先提升建议 |

## 六、AI成长规划中心

### 功能点
| 功能 | 说明 |
|------|------|
| 目标岗位选择 | 下拉框选择：AI算法工程师、AI Agent工程师、Java架构师、大数据工程师、全栈工程师、DevOps专家 |
| 7阶段成长路线 | 点击"生成成长路线"后，展示带时间线样式的完整学习路径 |
| 阶段详情 | 每阶段包含：阶段编号、标题、时长、学习内容描述、关联技能标签 |
| 资源推荐 | 底部推荐学习平台：Coursera、GitHub、知乎、CSDN、B站 |
| 总时长预估 | 自动计算全部阶段总周数 |

### 成长路线阶段（以AI方向为例，共40周）
| 阶段 | 内容 | 时长 |
|------|------|------|
| 基础夯实 | Python语法进阶、数据结构与算法、Git、Linux | 第1-4周 |
| 机器学习入门 | Scikit-learn、特征工程、模型评估 | 第5-10周 |
| 深度学习进阶 | PyTorch、Transformer、NLP/CV基础 | 第11-16周 |
| 大模型时代 | LLM微调、RAG技术栈、LangChain、Prompt | 第17-22周 |
| AI Agent实战 | Agent架构、CrewAI、MCP、GraphRAG | 第23-28周 |
| 项目实战 | 完整AI项目：需求→架构→开发→部署 | 第29-34周 |
| 面试冲刺与Offer | 刷题、系统设计面试、简历优化、谈薪 | 第35-40周 |

## 技术栈

| 技术 | 用途 |
|------|------|
| HTML5 + CSS3 | 页面结构 + 暗色主题设计系统 |
| JavaScript (原生) | 全部交互逻辑、数据处理、DOM操作 |

## 使用方式

直接在浏览器中打开 `index.html` 即可运行，所有数据均为内置静态数据，无需安装任何依赖或启动后端服务。
